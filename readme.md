This is a picasa web template for web export to a jekyll site.

The code it emits uses jekyll layouts so it will automatically have the theme and navigation of a site.

It uses layouts named: gallery and galleryitem. 

Place the folder in the Program Files\Google\Picasa3\web\templates folder.
